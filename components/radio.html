# Color

Use the color property to change the base color of the component and some of the child components, to better understand the handling of colors and themes you can see it here

Allowed values are:

  • primary
  • success
  • danger
  • warning
  • dark
  • RGB
  • HEX
Code Copied
  • Template
  • Script
  • All 
<template>
  <div class="center">
    <vs-radio v-model="picked" value="1"> Primary </vs-radio>
    <vs-radio v-model="picked" value="2" color="success"> Success </vs-radio>
    <vs-radio v-model="picked" value="3" color="danger"> Danger </vs-radio>
    <vs-radio v-model="picked" value="4" color="warn"> Warning </vs-radio>
    <vs-radio v-model="picked" value="5" color="dark"> Dark </vs-radio>
    <vs-radio v-model="picked" value="6" color="#7d33ff"> HEX </vs-radio>
    <vs-radio v-model="picked" value="7" color="rgb(59,222,200)">
      RGB
    </vs-radio>
  </div>
</template>

# Label New

Add a label to the radio with the default slot, if you need the label to be before the radio you can use the label-before property

Code Copied
  • Template
  • Script
  • All 
<template>
  <div class="center">
    <vs-radio v-model="picked" value="1"> Label </vs-radio>
    <vs-radio v-model="picked" label-before value="2"> label Before </vs-radio>
  </div>
</template>

# Loading New

Add a loading animation to the component, when the radio has this property active it is as if it were in disabled

Code Copied
  • Template
  • Script
  • All 
<template>
  <div class="center">
    <vs-radio v-model="picked" loading value="1"> Option 1 </vs-radio>
    <vs-radio v-model="picked" loading value="2"> Option 2 </vs-radio>
  </div>
</template>
